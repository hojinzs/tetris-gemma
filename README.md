# 🕹️ Tetris Gemma (Browser-based)

A fully functional, browser-based Tetris game built with HTML5 Canvas, CSS3, and JavaScript. This project was developed to demonstrate the software engineering capabilities of **Gemma 4**, a large language model running in a local environment.

## ✨ Features

- **Smooth Gameplay**: High-performance rendering using `requestAnimationFrame`.
- **Advanced Visuals**: Blocks feature a 3D-like appearance with highlights and shadows for depth.
- **Ghost Piece**: A transparent "ghost" block shows exactly where the current piece will land, aiding precision.
    - **Hard Drop**: Press `Space` to instantly drop the piece to the bottom.
- **Next Piece Preview**: See which tetromino is coming up next in the side panel.
- **Score & Line Tracking**: Keep track of your progress with real-time score and lines cleared updates.
- **Game States**: Smooth transitions between Start, Playing, and Game Over (with Try Again) states.

## ⌨️ Controls

| Key | Action |
| :--- | :--- |
| `←` / `→` | Move Left / Right |
| `↓` | Soft Drop (Move Down Faster) |
| `Space` | **Hard Drop** (Instant Drop) |
| `↑` or `W` | Rotate Clockwise |
| `Q` | Rotate Counter-Clockwise |

## 🚀 How to Run

No installation required! Since it's built with standard web technologies, you can run it directly in any modern web browser.

1. Download the repository files (`index.html`, `style.css`, `script.js`).
2. Open `index.html` in your preferred browser (Chrome, Edge, Firefox, etc.).
3. Click **[START GAME]** and enjoy!

## 🛠️ Technologies Used

- **HTML5 Canvas API**: For high-performance 2D rendering.
- **CSS3**: For modern, responsive UI styling and layout.
- **JavaScript (ES6+)**: To implement the core game engine, collision detection, and physics.
